## DB_LP実績 プロパティ

| プロパティ名 | 型 | 必須 | 用途 |
|---|---|---|---|
| キー | タイトル | ✓ | site.json の `key`(英数スネークケース。例: `coconala_rating`) |
| ラベル | テキスト | ✓ | カード下の日本語見出し(例: `ココナラ平均評価`) |
| 値 | テキスト | ✓ | 表示する数字(例: `4.9`, `30`, `99`) |
| 単位 | テキスト | - | `件+` `%` など。空欄可 |
| アイコン種類 | セレクト | ✓ | `star` / `chart` / `smile` |
| 表示順 | 数値 | ✓ | 1, 2, 3... |
| 公開 | チェックボックス | ✓ | ✓ のみ取得 |
| ticker用ラベル | テキスト | - | 空なら `ラベル + 値 + 単位` で自動生成 |

## DB_お客様の声 プロパティ

| プロパティ名 | 型 | 必須 | 用途 |
|---|---|---|---|
| 顧客名 | タイトル | ✓ | `A様` など |
| 性別属性 | テキスト | - | `女性` `男性 / 個人事業主` など |
| 評価 | 数値(1〜5) | ✓ | 星の数 |
| 本文 | テキスト | ✓ | カード本文 |
| 公開 | チェックボックス | ✓ | ✓ のみ取得 |
| 表示順 | 数値 | ✓ | 上位3件のみ表示 |
| ticker用ショート | テキスト | - | tickerに流す短文 |

## DB_LP改善提案 プロパティ(Self-Refinement対応)

| プロパティ名 | 型 | 用途 |
|---|---|---|
| タイトル | タイトル | `[観点] 短い改善提案見出し` |
| 観点 | セレクト | `マーケ` / `UX` / `コピー` / `競合参照` / `構成` |
| 重要度 | セレクト | `High` / `Mid` / `Low` |
| 影響範囲 | マルチセレクト | `ヒーロー` 等 |
| 提案内容 | テキスト | なぜ・何を・どう変えるか |
| 想定差分 | テキスト | Before / After |
| ステータス | セレクト | `提案中` / `承認` / `却下` / `実装済` / `差し戻し` |
| 自動実装可 | チェックボックス | テキスト/数字置換のみ✓ |
| **リビジョン** | **数値** | **Self-Refinementで何回推敲したか(1-3)** |
| **自己評価スコア** | **数値** | **refinerが付けた0-100の品質スコア** |
| **推敲ログ** | **テキスト** | **各リビジョンの自己批判と修正内容** |
| 差分プレビュー | テキスト | implementer実行後にgit diffを記録 |
| 作成エージェント | テキスト | エージェント名 |
| 関連競合LP | リレーション(DB_競合LP参考) | 参照したLPカード |
